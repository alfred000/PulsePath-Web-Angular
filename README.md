# PulsePath-Web-Angular
Interface graphique et Dashboard prédictif pour PulsePath Engine. Développé en Angular (TypeScript). Consomme les API REST .NET/NodeJS.

# 🎨 PulsePath Web - Interface Angular V2

Ce dépôt contient l'application cliente Angular qui sert d'interface visuelle pour le projet **PulsePath Engine** [1, 2].

## 🎯 Fonctionnalités Clés
- **Formulaire d'Onboarding & Saisie** : Capturation des logs de santé (Poids, Calories, Pas, Sommeil, Macros, Sport, Jeûne) [1].
- **Dashboard Visuel Dynamique** : Restitution instantanée des calculs métaboliques, de vélocité et des KPIs [1].
- **Alerte de Trajectoire & Correction Prescriptive** : Affichage graphique du plan de rattrapage en cas d'écart.

## 🛠️ Stack Technique
- Framework : Angular (v17+)
- Langage : TypeScript
- Communication : HttpClient (Consomme les endpoints REST /api/pulsepath)
